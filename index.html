Color Gallery

This project is a colorful showcase of primary and secondary colors, arranged in a responsive grid layout. Each figure represents a single color with a caption to help users identify it. The page adapts to different screen sizes using a mobile-first design approach.

As the screen size increases, the grid changes layout: one column on mobile, two columns on tablets, and three columns on larger screens. Accessibility features like descriptive alt text and skip links are included to make the site usable for everyone.
